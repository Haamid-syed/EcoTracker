# Carbonly Frontend

This is the React + TypeScript + Vite frontend for the Carbonly project.

## Development

We recommend running the project from the root directory using the monorepo scripts:

```bash
# In the root project directory:
npm run dev
```

To run only the frontend:
```bash
npm install
npm run dev
```

## 🌐 Cloud Deployment Disclosure

The deployed version of this application on Vercel is for **demonstration only**. Browsers block public websites from reading your private hardware telemetry. For real-time monitoring of your own machine, follow the **Development** steps above to run the system locally.
